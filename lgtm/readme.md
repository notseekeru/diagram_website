# Overview

This folder cointains the observability configuration for the diagram website project. It includes OpenTelemetry instrumentation for the backend, as well as configuration for logging, tracing, and metrics collection. The goal is to ensure that we have visibility into the performance and behavior of the application, allowing us to identify and troubleshoot issues effectively.

## Relevant Application Version

- Loki - 3.7.1
- Grafana - 13.0.1
- Tempo - 2.6.0
- Prometheus 3.11.3
- Alloy 1.16.2.
